## Instalation 
1. Install dependencies
```
python3 -m pip install --user -r requirements.txt
```
**For Linux/BSD/OSX**: depending on your setup you may need to run
```
sudo python3 -m pip install -r requirements.txt
```


2. Run with:
```
python3 quiz_app.py
```
